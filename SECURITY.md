# Security Policy

## Reporting a Vulnerability

Please report vulnerabilities to caldera@mitre.org

Please include:
- Affected version
- Steps to reproduce
- (optional) suggested fix

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| >= 2.9.x  | :white_check_mark: |
| < 2.9   | :x:                |
